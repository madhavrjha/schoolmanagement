Basics

Some information like your academic year, and the current grade, and the grade for which you want to apply. all are mandatory to fill.

Profile

There is some personal details of student like first name and last name, and email should be valid, phone number must be without country code and starts with zero(0), and gender and dob. contact details of child are optional.

Primary Parent

Primary parent is will be the guardian of the learner. and there are some details like, name of the parent and email and phone of the parent. email must be valid and phone number should be without country code and must starts with zero(0), and Occupation, national identification number and, dob.

Secondary Parent

Secondary parent would be guardian when primary parent is not available. there are some details like, name of the parent and email and phone of the parent. email must be valid and phone number should be without country code and must starts with zero(0), and Occupation, national identification number and, dob.

Documents

There would be your address. and some documents to upload like, learner photo, birth Certificate, and two more fields to upload document.
